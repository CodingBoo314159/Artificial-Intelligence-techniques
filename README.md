# 🤖 Artificial Intelligence Techniques Project

## 📌 Overview
This project explores multiple machine learning and deep learning techniques applied to real-world classification problems. It includes models for:

- Customer churn prediction (Logistic Regression)
- Flood risk classification (Neural Network)
- News headline classification (NLP with Deep Learning)

The goal is to compare classical machine learning methods with modern neural network approaches.

---

## 📁 Project Structure

Artificial-Intelligence-techniques/

├── churn_model.py        # Logistic Regression model for churn prediction  
├── flood_model.py        # Neural network for flood risk classification  
├── nlp_model.py          # Deep learning model for text classification

├── flood_dataset.csv     # Dataset for flood risk model

## 📊 Models Implemented

### 1. Customer Churn Prediction
- Logistic Regression  
- StandardScaler preprocessing  
- Accuracy and Precision evaluation  

### 2. Flood Risk Classification
- Neural Network (Dense layers)  
- Dropout + Early Stopping  
- Precision-based evaluation  

### 3. News Headline Classification
- Embedding-based neural network  
- Tokenization + padding  
- Fake vs Real classification
